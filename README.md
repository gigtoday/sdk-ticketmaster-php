# Ticketmaster PHP SDK

PHP SDK for the [Ticketmaster Open Platform](http://developer.ticketmaster.com/).

## Usage

```bash
# PHP >= 7.1.3
composer require matchilling/sdk-ticketmaster:2.0.0

# PHP 5.6
composer require matchilling/sdk-ticketmaster:1.0.0
```
